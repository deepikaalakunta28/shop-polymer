# SHOP-POLYMER-WEBSITE

- A responsive Shop-Polymer-Website built using HTML and CSS, supporting both mobile-first and desktop layouts. 
- The project focuses on clean layout structure, desktop alignment, and flexible mobile views.


## Project Structure

```
shop-polymer-project/
│
├── home.html
├── pages/
│ ├── mens-outerwear.html
| ├── ladies-outerwear.html
│ ├── each-item.html
│ ├── cart.html
│ └── checkout.html
│
├── styles/
│ ├── reset.css
│ └── main.css
│
├── images/
│ └── *.jpg
│
└── README.md
```

## Pages

- **mens-outerwear :**  Mens outerwear listing page
- **ladies-outerwear :**  Ladies outerwear listing page
- **each-item :** Individual product details page
- **add-card :** Shopping cart page
- **checkout :** Checkout page


## Technologies Used

- **HTML5** – Page structure

- **CSS3** – Flexbox & Grid layout
